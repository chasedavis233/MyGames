# ColorStack (Tetris Clone)

Stack colorful falling blocks and clear lines!

- Smooth falling blocks
- Rotation and movement controls
- Line clearing and scoring
- Fast-paced as you survive longer
- Resizable window

## Controls
- **Left Arrow**: Move left
- **Right Arrow**: Move right
- **Up Arrow**: Rotate block
- **Down Arrow**: Drop faster
- **Spacebar**: Hard drop (instant)

## Requirements
- Python 3.x
- Pygame
