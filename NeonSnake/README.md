# NeonSnake

A neon-glowing Snake game where you grow as you eat food!

- Futuristic grid design
- Smooth snake movement
- Colorful food
- Resizable window
- Fun scoring system

## How to Play
- Use **Arrow Keys** to move the snake.
- Eat the neon food to grow.
- Avoid running into yourself or the walls!

## Requirements
- Python 3.x
- Pygame
